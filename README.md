# SellerGrid - Full Stack E-commerce Review Platform

## Features
- Dashboard: stats cards, sentiment chart, key issues
- Reviews page with negative reviews highlighted
- AI Insights page
- Multi-page structure

## Tech Stack
- Frontend: HTML, CSS, JS
- Backend: Node.js + Express
- Data: JSON
